# A/B Test Analytics (SQL & Tableau)

## Project Overview
This project analyzes A/B test performance using session, event, and account creation data.

The analysis focuses on user behavior, engagement, and conversion metrics across test groups.

## Tools Used
- SQL
- Tableau Public

## Analysis Areas
- Session analysis
- Event tracking
- Conversion analysis
- New account creation
- Test group comparison

## SQL Techniques
- CTEs
- Joins
- Aggregations
- UNION ALL
- Distinct counting

## Key Metrics
- Sessions
- Sessions with orders
- Events
- New accounts
